# AI Sentinel Dashboard

Modern, high-fidelity dashboard for the Multi-Modal Deepfake Detection System.

## Features
- Sleek Dark Mode UI with Glassmorphism
- Image and Video Upload Support
- Real-time Inference Status
- Frame-wise Probability Visualization
- Grad-CAM Heatmap Analysis
- PDF and JSON Report Export

## Tech Stack
- React
- Typescript
- Tailwind CSS
- Shadcn UI
- Lucide React
- Recharts
- Framer Motion

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Navigate to the dashboard directory:
   ```bash
   cd ai-sentinel-dashboard
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
To build for production:
```bash
npm run build
```
